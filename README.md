# IT Ticket Analysis (SLA & Resolution Insights)

## 📌 Project Overview
This project analyzes IT support tickets to evaluate SLA compliance, resolution time, and recurring issues. The goal is to improve support efficiency and identify operational bottlenecks.

## 🛠 Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- SQL
- Excel

## 📊 Key Analysis Performed
- SLA compliance calculation
- Resolution time analysis
- Ticket volume trends by category and priority
- Identification of recurring issues

## 📈 Key Insights
- Identified categories with highest ticket volume
- Highlighted SLA breaches and delay patterns
- Found recurring issues causing operational inefficiencies
- Suggested improvements to reduce resolution time

## 📁 Files Included
- tickets_data.csv
- ticket_analysis.py / .ipynb
- visualizations.png

## 🚀 Business Impact
This analysis helps IT teams improve response time, meet SLA targets, and optimize support processes.
